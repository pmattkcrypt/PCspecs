Μας επιστρέφει τα βασικά χαρακτηριστικά ενός συστήματος Linux (RAM,CPU,H.D.D.,O/S) 
Για κάθε σύστημα δημιουργεί ένα αρχείο με όνομα PCspecs_όνομα της συσκευής.

-------------------------------------
Οδηγίες:
1) Κατεβάζετε το script.
2) Ανοίγετε το Τερματικό.
3) Αντιγράφετε το script από της λήψεις στο φάκελο του τερματικού (Προσωπικός Φάκελος).
4) Την πρώτη φορά που θα τρέξετε το script, πρέπει να του δώσετε δικαιώματα εκτέλεσης, $chmod +x pcspecs.sh
5) Εκτέλεση του με $./pcspecs.sh
6) Την πρώτη φορά που θα τρέξει ζητάει κωδικό για την εκτέλεση. Δίνετε τον κωδικό του συστήματος.
7) Εμφανίζονται τα χαρακτηριστικά και δημιουργείται το αντίστοιχο αρχείο στον προσωπικό φάκελο του τερματικού με όνομα PCspecs_όνομα της συσκευής.
   

Σημείωση:
Το script δοκιμάζεται σε διάφορα συστήματα Linux, ώστε να γίνουν οι απαραίτητες διορθώσεις αλλά και τυχόν επεκτάσεις, όπως π.χ το αρχείο που προκύπτει (PCspecs_όνομα) να αντιγράφετε στο Server ή σε έναν άλλο σύστημα.
Με $nano pcspecs.sh μπορείτε να δουλέψετε το script όπως εσείς θέλετε και να το προσαρμόσετε.
